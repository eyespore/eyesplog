# Java

## 命名规范

- 项目名：

  Java项目名全部采用**小写字母**，多个单词时采用中划线`-`来进行分隔
  例如：`my-java-project`

  ***

- 包名：

  包名也采用**小写字母**，使用点`.`作为单词分隔符，包名通常由项目发起人、项目名、模块名等部分组成，具体取决于公司命名策略
  例如：`com.example.project`

  ***

- 类名：

  采用大驼峰书写
  例如：`HelloWorld.java`

  ***

- 变量名：

  采用小驼峰书写

  例如：`fileContext`

  ***

- 常量名：

  全大写，使用下划线分隔单词
  例如：`STATIC_VALUE`
